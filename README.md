# 1lt_config
A simple to read, edit and parse config file

## Example:
<code>

[files]
save_file_to = /home/example_user/.local/share/example_application/save_file.txt
read file from=/home/example_user/.local/share/example_application/read_file.txt

[ui]
hide_this = true
hide_that = false
size_of_that = 3
</code>
